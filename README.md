# UnniTestFramework
unnitest框架代码

# 搭建测试框架
    apiTestFramework       项目名称
    一 api                 定义封装被测系统的接口
    一 script              定义测试用例脚本
    一 data                存放测试数据
    一 report              存放生成的测试报告
    一 lib(tools)          存放第三方的文件
    一 app.py              定义项目的配置信息
    一 utils. py           定义工具类
    一 run_ suite.py       执行测试套件的入口
